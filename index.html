Tentang Platform Nexus IoT

Sistem monitoring canggih yang menghubungkan perangkat ESP32/ESP8266 ke Firebase. Dirancang untuk kemudahan instalasi, dashboard elegan menampilkan data sensor suhu, kelembaban, tekanan, dan perangkat IoT lainnya secara realtime. Cocok untuk proyek smart agriculture, industri, maupun riset.

  • Integrasi plug-and-play dengan ESP32 / Arduino
  • Mendukung berbagai sensor (DHT11, DS18B20, BME280)
  • Dashboard modern & dark mode
  • Sinkronisasi realtime tanpa refresh halaman
